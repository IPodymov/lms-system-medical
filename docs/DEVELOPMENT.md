# Разработка

Локальные настройки находятся в `.env`; файл готов к запуску с SQLite. После изменения моделей: `python manage.py makemigrations && python manage.py migrate`. Compose активирует PostgreSQL.

## Проверка конструктора курса

После изменения конструктора проверьте создание черновика, его появление только у автора в каталоге, добавление темы и блока, создание теста на отдельной странице и добавление внешней ссылки. Примените миграции до открытия редактора: в частности, миграция `courses.0003_coursemateriallink` создаёт таблицу ссылок на дополнительные материалы.

## Документация администратора

Страница документации выдаётся только суперпользователю из панели «Управление». Не добавляйте постоянную публичную ссылку: URL формируется на каждый просмотр панели, содержит подписанный токен и проверяется сервером.
