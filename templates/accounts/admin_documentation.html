Документация администратора

Страница доступна только системному администратору по подписанной ссылке.

Рабочий процесс курса

  1. Преподаватель вводит название и описание — система создаёт черновик.
  2. Черновик появляется в каталоге только у автора и ведёт в конструктор программы.
  3. В конструкторе добавляются разделы, темы, текстовые блоки, файлы, тесты и дополнительные ссылки.
  4. Кнопка «Открыть черновик курса» позволяет автору проверить страницу курса до публикации.
  5. При публикации система создаёт активный основной поток, открывает запись и показывает курс слушателям в общем каталоге.

Обслуживание

После обновления кода применяйте миграции: python manage.py migrate. Для проверки: python manage.py test, ruff check ., ruff format --check ..

Безопасность

Ссылка подписана ключом Django, действует 24 часа и связана с идентификатором открывшего её суперпользователя. При передаче ссылки другой учётной записи доступ не предоставляется.
